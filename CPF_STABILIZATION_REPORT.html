실패/교정

항목내용
PowerShell pipe 방식 MariaDB 실행실패. SQL 파일을 PowerShell pipe로 native client에 넘기면 한글 seed가 ?로 변환되는 것을 HEX 조회로 확인했습니다. 파일 리다이렉션 방식으로 재실행해 한글 저장을 교정했습니다.

미검증

항목사유
OpenAPI HTTP smoke미검증. ADM/XYZ/BIZADM/EXS 앱을 장기 기동하지 않았으므로 /v3/api-docs HTTP 검증은 성공으로 기록하지 않습니다.
브라우저 클릭 검증미검증. Playwright 같은 브라우저 자동화로 실제 클릭을 수행하지 않았습니다. 정적 ADM UI smoke만 수행했습니다.
VSCode Problems 0건미검증. IDE Problems 패널은 CLI에서 직접 조회하지 못했습니다. compile/test/qualityGate 결과만 기록합니다.
Redis/Kafka 실 broker미검증. 실제 broker를 기동하지 않았습니다. DB fallback 기준만 코드/문서에 유지했습니다.
TOTP 앱 QR/복구코드미검증. 이번 범위에서는 앱 호환 OTP 등록 UI/API를 실제 구현/기동 검증하지 않았습니다.
기존 운영 DB 증분 migration미검증. V6__bizadm_exs_transaction_identity.sql 파일은 추가했지만, 별도 기존 운영 DB 상태에서 Flyway로 적용 검증하지 않았습니다.
