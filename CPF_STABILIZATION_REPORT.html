CPF 안정화 작업 리포트

요청서 기준으로 trusted proxy invalid IP 검증, header mutator 테스트, EDU Mapper slice 존재/실행 상태, 리포트와 가이드 정합성을 보정했습니다.

이번 작업에서는 MariaDB 신규 DB 전체 설치, ADM runtime/OpenAPI, 실제 브라우저 클릭, Redis/Kafka/MQ 실 broker, 센터컷, 온디맨드 배치, CMN 파일 공통 기능을 실행하거나 구현하지 않았습니다. 실행하지 않은 항목은 성공으로 기록하지 않았습니다.

미검증과 완료 불인정

대상상태사유다음 조치완료로 기록하지 않은 이유
EDU Mapper DB fixture 실행미검증안전한 테스트 DB 환경변수 CPF_XYZ_EDU_MAPPER_SLICE_TEST, DB URL, user, password가 없어 DB 실행 테스트는 skip되었습니다.안전한 빈 테스트 DB를 준비하고 환경변수를 지정한 뒤 .\gradlew.bat :xyz:test --offline를 실행합니다.skip은 성공이 아니므로 DB fixture 실행 완료로 기록하지 않습니다.
MariaDB 신규 DB 전체 설치미검증이번 요청 제외 범위이며 cleanup 위험이 있습니다.안전한 빈 DB 또는 사용자 명시 승인 DB에서 전체 설치와 seed idempotent를 검증합니다.실제 SQL 실행을 하지 않았습니다.
ADM runtime/OpenAPI미검증이번 요청 제외 범위입니다.ADM 기동 후 health, OpenAPI JSON, 주요 API smoke를 별도 수행합니다.앱 기동과 runtime API 호출을 하지 않았습니다.
ADM 실제 브라우저 클릭미검증이번 요청 제외 범위입니다.브라우저 자동화 또는 수동 절차로 주요 화면과 버튼을 검증합니다.정적 파일 검사를 실제 클릭 성공으로 간주하지 않습니다.
Redis/Kafka/MQ 실 broker실환경 필요이번 요청 제외 범위이며 실 broker가 필요합니다.mock/fallback과 실 broker 검증을 분리해 진행합니다.실 broker 전파 검증을 수행하지 않았습니다.
