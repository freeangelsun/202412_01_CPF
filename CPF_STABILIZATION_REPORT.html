FK / Index / Comment / 공통 감사 컬럼 검증

요청 대상 테이블은 모두 존재했습니다. 모든 대상 테이블에 created_by, created_at, updated_by, updated_at 4개 감사 컬럼이 존재했고, 컬럼 comment 누락은 0건이었습니다. FK가 0건인 독립 기준 테이블은 참조를 받는 기준 테이블 성격으로 확인했습니다.
