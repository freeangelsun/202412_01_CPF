CPF 안정화 작업 리포트

ChatGPT 검수 결과로 지정된 5개 항목을 기준으로 실제 파일 불일치와 미완성 증거를 보정했습니다. 이번 작업은 신규 대형 기능 확장이 아니라 trusted proxy 연결, user/operator mutator 차단, EDU Mapper slice 구조, SQL 설치 정합성 gate, 리포트/매트릭스 정합성에 한정했습니다.

실행하지 않은 MariaDB 신규 DB 설치, ADM runtime/OpenAPI, 브라우저 클릭, Redis/Kafka/MQ 실 broker 검증은 성공으로 기록하지 않았습니다. skip된 EDU DB slice test도 성공으로 기록하지 않았습니다.

수정한 기능과 파일

요청수정한 파일실제 변경 내용상태
Trusted Proxy 실제 연결CpfHeaderExtractor, CpfTrustedProxyPolicy, LoggingAspect, CpfHeaderStandardCoverageTest, 표준_헤더_가이드.htmlclient IP 산정이 CpfTrustedProxyPolicy.resolveClientIp를 사용하도록 연결했습니다. trusted proxy가 아니면 전달 IP 헤더를 무시하고, trusted proxy일 때만 X-Forwarded-For, Forwarded, X-Real-IP를 검증된 IP 값 기준으로 해석합니다.반영
User/Operator Mutator 차단CpfHeaderMutator, CpfHeaderMutatorTest, 표준_헤더_가이드.htmlX-User-Id, X-Operator-Id, correlation, idempotency key 등 추적/감사 주체성 헤더를 restricted에 포함했습니다. 고객/회원/테넌트/조직/지점/세부채널은 업무 보정 가능 후보로 유지했습니다.반영
EDU Mapper slice 구조XyzQueryEducationMapperSliceTest, xyz_edu_query_mapper_fixture.sql, XyzQueryEducationRepositoryTest, 개발_가이드.htmlMyBatis XML을 읽어 unsafe ${}, whitelist 정렬, 직접 JOIN 금지를 기본 test로 확인합니다. 안전한 테스트 DB 환경변수가 있을 때 fixture SQL로 실제 Mapper XML을 실행하는 선택 실행 test 구조를 제공합니다.반영
SQL 정합성check-sql-standard.ps1, SQL_가이드.html, SQL 설치본cmn_edu_query_item이 split SQL, V14, baseline, 00_all_install.sql, 00_all_install_and_smoke.sql에 존재함을 확인했습니다. split schema 테이블이 전체 설치본과 baseline에 누락되면 gate가 실패하도록 검사 로직을 추가했습니다.반영
리포트/매트릭스 정합성CPF_STABILIZATION_REPORT.html, 기능_구현_매트릭스.html구현됨과 runtime/브라우저/실DB 검증됨을 분리했습니다. 정적 ADM UI marker와 실제 브라우저 클릭을 별도 상태로 유지했습니다.반영

미검증과 완료로 인정하지 않은 항목

대상시도한 작업상태값완료로 기록하지 않은 이유다음 조치
EDU Mapper DB slice test선택 실행 test와 fixture SQL을 추가하고 기본 test에서 skip 여부를 확인했습니다.미검증안전한 테스트 DB 환경변수가 없어 실제 fixture SQL과 MyBatis XML DB 실행부는 skip되었습니다.안전한 빈 테스트 DB를 준비하고 CPF_XYZ_EDU_MAPPER_SLICE_TEST=true, DB URL/user/password를 지정해 :xyz:test를 실행합니다.
MariaDB 신규 DB 전체 설치SQL 파일과 gate 기준을 정적 검증했습니다.미검증00_all_install_and_smoke.sql는 cleanup 위험이 있어 승인 없이 실제 DB에 실행하지 않았습니다.안전한 빈 DB 또는 명시 승인된 DB에서 전체 설치, seed 재실행, app/migration 권한 분리를 검증합니다.
ADM runtime/OpenAPI이번 범위에서 제외하고 문서/리포트에 미검증으로 유지했습니다.미검증요청서에서 신규 runtime smoke 구현은 제외 범위입니다.ADM 기동 후 health, OpenAPI JSON, 주요 API smoke를 별도 작업으로 실행합니다.
ADM 브라우저 클릭정적 UI marker와 실제 브라우저 클릭 상태를 분리했습니다.미검증정적 marker smoke를 브라우저 클릭 성공으로 보지 않습니다.브라우저 자동화 또는 수동 검증 절차로 메뉴, 버튼, 로그 상세 탭을 확인합니다.
Redis/Kafka/MQ 실 broker이번 범위에서 제외하고 추후 이슈로 분리했습니다.실환경 필요실 broker와 장애 시나리오가 필요합니다.broker 우선 전파와 DB fallback 장애 시나리오를 별도 검증합니다.
