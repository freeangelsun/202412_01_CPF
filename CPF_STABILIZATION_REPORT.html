남은 리스크

  • BIZADM/MBR/EXS DB repository와 SQL 구조는 정적 검증됐지만, MariaDB 실실행은 client 부재로 미검증입니다.
  • ADM 정적 smoke는 통과했지만 실제 브라우저 클릭, 권한별 버튼 비활성 사유, 화면 렌더링은 미검증입니다.
  • OpenAPI smoke 스크립트는 전 모듈 검사 구조로 확장했지만, 앱 기동 후 HTTP 검증은 미실행입니다.
  • Spring Batch JobRepository, JobExplorer, JobOperator는 컴파일 기준 연결됐지만, 실 DB Job 실행 시나리오는 미검증입니다.
  • 실제 OTP 앱 QR, 복구코드, Redis/Kafka 실 broker 연동은 이번 범위에서 구현 검증하지 않았습니다.
