CPF PRODUCT DOCUMENTATION SET

사용 방법
1. README.md를 Repository Root의 기존 README.md와 교체
2. cpf-docs 디렉터리를 Repository Root에 복사
3. PRE_CODEX_CLEANUP_CHECKLIST.md를 읽고 Root를 정리
4. 정리 후 PRE_CODEX_CLEANUP_CHECKLIST.md와 _PACKAGE_README.txt는 Repository에 넣지 않음
5. Codex 작업 시작 전 Git status 확인

주의
- 이 문서는 최종 제품 구조를 기준으로 작성한 완성형 문서 세트입니다.
- 실제 Task명, Port, 설정 Key와 Script 경로는 Codex가 최종 Source와 일치하도록 검증·보정해야 합니다.
- 기존 legacy Module을 사용자가 먼저 삭제하지 마십시오.
