CPF SQL 가이드

CPF의 설치 SQL, Flyway migration, seed, smoke query, DB 표준을 정리합니다. DB 표준 세부 기준은 DB 표준 가이드에서 함께 확인합니다.

DB 소유권

DB소유 영역주요 테이블주의
pfwDB프레임워크 코어와 운영 메타pfw_transaction_log, pfw_log_policy, pfw_message, pfw_response_code, pfw_config, BATCH_*, pfw_batch_*, pfw_center_cut_*업무 데이터 저장소로 사용하지 않습니다.
cmnDB프로젝트 업무 공통cmn_sequence, cmn_notification_log, cmn_business_log, cmn_edu_query_item프로젝트에서 필요할 때 optional datasource로 붙입니다.
admDB운영자 권한과 감사adm_operator, adm_role, adm_menu, adm_button, adm_api_permission, adm_audit_log운영 행위 변경은 감사 사유와 before/after diff를 남깁니다.
mbrDB회원 업무mbr_member, mbr_member_role, mbr_member_login_history, mbr_refresh_tokenADM은 운영 API로 관리하지만 회원 데이터 소유권은 MBR에 있습니다.
bizadmDB, exsDB업무 관리자, 외부 연계 예제bizadm_*, exs_*업무 영역 prefix와 공통 감사 컬럼을 유지합니다.

설치 SQL

파일역할기준
specs/sql/00_all_install.sql전체 설치 합본SOURCE 지시자가 아니라 실제 SQL 본문을 포함합니다.
specs/sql/00_all_install_and_smoke.sql전체 설치와 smoke query 합본cleanup을 포함하므로 기존 DB에 실행하면 데이터가 삭제될 수 있습니다.
specs/sql/10_pfw_schema.sqlPFW schema프레임워크 코어, Spring Batch, CPF Batch 운영 메타, center-cut 표준 메타를 포함합니다.
specs/sql/50_framework_seed_data.sqlPFW seed코드, 메시지, 응답코드, 설정, batch job, center-cut 모수를 idempotent 방식으로 적재합니다.
scripts/build-all-install-sql.ps1합본 SQL 재생성분리 SQL 변경 후 합본과 Flyway baseline을 동기화합니다.

center-cut SQL 기준

테이블역할확장 기준
pfw_center_cut_jobcenter-cut Job 모수PFW/BAT 공통 실행 기준과 ADM 관제 기준을 제공합니다.
pfw_center_cut_parametercenter-cut 실행 파라미터secret 값은 직접 저장하지 않고 참조 키 또는 암호화 정책을 사용합니다.
pfw_center_cut_item표준 대상 상태업무별 대용량 대상 테이블이 필요한 경우 업무 DB에 별도 target/item/result 테이블을 만들고 adapter로 연결합니다.
pfw_center_cut_result표준 결과 이력부모/자식 거래 글로벌 ID와 결과 상태를 운영 추적 기준으로 남깁니다.
