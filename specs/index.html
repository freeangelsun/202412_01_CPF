CPF CoreFlow Platform Framework 문서 인덱스

이 문서는 개발자, 운영자, 검수자가 같은 기준으로 CPF 프레임워크를 확인하도록 만든 진입점입니다. README는 짧은 개요로 유지하고, 상세 기준은 HTML 가이드 문서에서 관리합니다.

역할 경계는 고정 기준입니다. PFW는 프레임워크 코어, CMN은 프로젝트 공통, BAT는 업무 배치 실행 worker, ADM은 운영 관제 콘솔입니다.

거래 추적 기준

X-Transaction-Id는 CPF 온라인 API와 운영 API에서 사용하는 34자리 거래 추적 ID입니다. 구조는 yyyyMMddHHmmssSSS 17자리, module ID 3자리, WAS ID 7자리, sequence 7자리 조합입니다. 호출자가 전달하지 않으면 PFW가 생성하고, 응답 헤더와 거래 로그에 같은 값을 전파합니다.
