현재 표준 요약

테이블명{주제영역}_{업무명} lower snake case. 예: pfw_message, cmn_sequence, adm_menu, mbr_member
공통 컬럼모든 테이블에 created_by, created_at, updated_by, updated_at 적용
SQL 주석공식 schema SQL의 모든 테이블/컬럼은 한글 COMMENT 작성
거래 헤더필수 헤더 3종과 클라이언트/호출자/멱등성 표준 선택 헤더를 Swagger, Context, 로그에 일치시킴
ADM 운영회원관리, 권한 매트릭스, 통합 로그 포맷팅, 배치 관제, 캐시/메시지/코드/설정 관리를 한 화면에서 운영
CMN DB기본은 비활성, cpf.cmn.business-db.enabled=true일 때 채번/알림/업무 로그 API 사용
