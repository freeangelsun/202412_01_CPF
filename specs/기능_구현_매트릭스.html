기능 구현 매트릭스

CPF 기능이 실제 소스, API, ADM 화면, DB, Swagger, EDU 샘플, 검증 명령과 연결되어 있는지 확인하기 위한 검수 기준입니다. 완료는 실제 실행 검증까지 확인된 경우에만 사용하고, 코드나 문서만 있는 기능은 일부 구현 또는 미검증으로 표시합니다.

검수 메모

  • 일부 구현은 소스 또는 SQL은 있으나 앱 기동, 브라우저 클릭, 실제 DB 적재, OpenAPI JSON 검증 중 하나 이상이 부족한 상태입니다.
  • 기능을 보강할 때는 주요 소스, API, ADM 화면, DB 테이블, Swagger, EDU 샘플, 테스트, 가이드를 함께 갱신해야 합니다.
  • 실행하지 않은 검증은 성공으로 표시하지 않고 미검증 또는 리포트의 보류 항목으로 분리합니다.
