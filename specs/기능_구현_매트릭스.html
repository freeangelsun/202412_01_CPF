CPF 기능 구현 매트릭스

이 문서는 기능별 구현 상태를 실제 파일, API, DB, Swagger, EDU, 테스트 증거와 함께 대조합니다. 부분 구현과 미검증 항목은 완료로 표시하지 않습니다.

기능별 상태

영역기능상태주요 증거검증남은 리스크
PFW거래 ID와 표준 헤더구현TransactionIdGenerator, TransactionContextFilter, LoggingAspect컴파일, transaction ID 표준 검사, MariaDB smoke운영 로그 데이터는 실제 운영일 기준으로 지속 점검해야 합니다.
PFW표준 응답과 예외부분 구현CpfGlobalExceptionHandler, pfw_response_code, pfw_message단위 테스트, OpenAPI smoke공통 오류 schema와 예외 케이스별 통합 테스트를 계속 보강해야 합니다.
PFWBatch 공통 API부분 구현CpfBatchLauncher, CpfBatchOperationRepository, pfw_batch_*, BATCH_*MariaDB smoke, ADM runtime smoke실시간 heartbeat listener, ghost 자동 감지, 실제 JobRepository 부하 시나리오가 다음 단계입니다.
CMN프로젝트 공통 채번/알림/로그부분 구현CmnSequenceService, CmnNotificationLogService, CmnBusinessLogServiceCMN 테스트, 동시성 테스트프로젝트별 보존 정책과 업무별 확장 규칙은 적용 프로젝트에서 확정해야 합니다.
ADM운영자 권한부분 구현AdmApiAuthFilter, AdmPermissionController, adm_role, adm_button정적 UI smoke, ADM runtime smoke서버 권한 차단 통합 테스트와 버튼 비활성 사유 UX 검증이 남아 있습니다.
ADM회원 관리부분 구현AdmMemberController, AdmMemberOperationService, mbr_member, mbr_member_role정적 UI smoke, MariaDB smoke회원 등록/수정/상태 변경 전체 E2E 검증이 필요합니다.
ADM로그 관제부분 구현AdmLogController, AdmLogQueryService, pfw_transaction_log정적 UI smoke, MariaDB smokeJSON pretty print, 고정길이 전문 포맷팅, 원문 보기 권한 클릭 검증이 필요합니다.
ADM배치 관제부분 구현AdmBatchController, AdmBatchOperationService, PfwBatchScheduler, pfw_batch_*MariaDB smoke, ADM runtime smoke관계도 그래프, 브라우저 클릭, 실시간 listener 갱신은 다음 단계입니다.
ADM기동 health와 runtime smoke구현AdmHealthController, scripts/smoke-adm-runtime.ps1health, OpenAPI, 배치 API, UI smoke브라우저 클릭 자동화는 미검증입니다.
BIZADM업무 관리자 기본 구현부분 구현BizAdmAuthController, BizAdmOperationController서비스 테스트, MariaDB smokeController부터 DB까지 이어지는 통합 테스트를 추가해야 합니다.
MBR회원 업무 구현부분 구현MbrController, MbrAuthController, MemberMapper서비스 테스트, MariaDB smoke회원 상태 전환, 탈퇴/복구, 권한 만료 통합 테스트가 필요합니다.
EXS대외 연계 구현부분 구현ExsOperationController, ExsFlowController, exs_*서비스 테스트, MariaDB smoke실제 기관 adapter, token refresh, retry worker 검증이 필요합니다.
XYZ EDU교육 샘플부분 구현xyz/src/main/java/cpf/xyz/edu컴파일, OpenAPI smoke배치 유형별 샘플과 학습용 주석은 계속 보강해야 합니다.
SQL/Flyway설치와 migration구현specs/sql, Flyway V1부터 V9까지 migrationSQL 표준 검사, MariaDB 전체 설치, seed 재실행, 권한 분리, FK/index/comment/감사 컬럼 smokeGradle Flyway task는 아직 등록되어 있지 않아 파일과 실제 DB 기준으로 대체 검증했습니다.
Swagger/OpenAPIAPI 문서부분 구현Controller annotation, PfwOpenApiAutoConfigurationADM runtime smoke에서 OpenAPI smoke 통과OpenAPI JSON 품질 gate를 CI 산출물로 고정하는 작업이 남아 있습니다.
