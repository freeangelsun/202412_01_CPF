CPF 기능 구현 매트릭스

기능 구현 상태와 검증 상태를 분리해 기록합니다. 실행하지 않은 검증은 완료로 기록하지 않습니다.

상태 값은 완료, 부분 구현, 미구현, 미검증, 실패, 재확인 필요만 사용합니다.

상태 기준

완료코드, SQL, 문서, 테스트 또는 smoke 증거가 현재 저장소 기준으로 맞습니다.
부분 구현핵심 구조는 있으나 runtime, 브라우저, 외부 연동, 운영 시나리오 중 일부가 남았습니다.
미구현구현 파일이나 동작 경로가 없습니다.
미검증구현 또는 스크립트는 있으나 이번 작업에서 실행하지 않았습니다.
실패이번 작업에서 실행했지만 실패했습니다.
재확인 필요환경 차이 또는 외부 조건 때문에 별도 확인이 필요합니다.

기능 상태

영역기능상태증거남은 확인
PFW표준 헤더와 거래 컨텍스트부분 구현CpfHeaderNames, CpfHeaderSpecs, CpfInboundHeaderValidator, CpfRestClientInterceptorRuntime E2E와 ADM 로그 상세 조회는 별도 smoke가 필요합니다.
PFW거래 로그와 로그 정책부분 구현LoggingAspect, LogPolicyResolver, pfw_log_policy, scripts/smoke-log-policy-runtime.ps1다중 인스턴스 broker 전파는 미검증입니다.
PFW/BATBatch 공통 API와 CPF Batch 메타부분 구현CpfBatchLauncher, CpfBatchOperationRepository, CpfBatchHeartbeatService, pfw_batch_*, BATCH_*운영 장애 시나리오와 ghost 자동 scheduler가 남았습니다.
PFW/BATcenter-cut 1차 계약과 BAT sample부분 구현CpfCenterCutService, CenterCutTargetProvider, CenterCutHandler, BatCenterCutSmokeTasklet, pfw_center_cut_*, CPF_BAT_CENTER_CUT_JOB업무 DB 기반 adapter와 ADM center-cut 상세 관제가 다음 단계입니다.
ADM운영자/권한/회원/로그/배치/캐시/메시지/코드/설정 API부분 구현AdmPermissionController, AdmMemberController, AdmLogController, AdmBatchController, adm/src/main/resources/static/adm/adm.js브라우저 클릭 UX와 일부 운영 화면 고도화가 남았습니다.
CMN프로젝트 공통 채번, 알림, 업무 로그, 파일/전문/MQ 보조부분 구현CmnSequenceService, CmnNotificationLogService, CmnBusinessLogService, CmnTelegramService프로젝트별 정책 확장과 DB 동시성 재검증이 필요합니다.
XYZ EDU조회/CRUD/CMN/배치 교육 샘플부분 구현XyzQueryEducationController, xyz_edu_query_fixture.sql, XyzBatchEducationControllercenter-cut EDU 샘플은 다음 단계입니다.
SQLsplit SQL, all install, Flyway migration부분 구현00_all_install.sql, 00_all_install_and_smoke.sql, V16__batch_center_cut_standard.sql실제 MariaDB full install은 이번 작업에서 별도 실행 여부를 리포트에 기록합니다.
