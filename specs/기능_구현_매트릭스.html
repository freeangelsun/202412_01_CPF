CPF 기능 구현 매트릭스

이 문서는 현재 저장소 기준 기능 구현 상태와 검증 상태를 분리해 기록합니다. 실행하지 않은 검증은 완료로 표시하지 않습니다.

상태 값은 완료, 부분 구현, 미구현, 미검증, 실패, 재확인 필요만 사용합니다. `CPF_STABILIZATION_REPORT.md`의 검증 표와 반드시 같은 check id, 같은 상태를 유지합니다.
