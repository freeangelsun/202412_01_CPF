CPF 기능 구현 매트릭스

현재 저장소 기준 구현 상태와 검증 상태를 분리해서 기록합니다. 실행하지 않은 검증은 완료로 표시하지 않습니다.

상태 값은 완료, 부분 구현, 미구현, 미검증, 실패, 재확인 필요만 사용합니다.

상태 기준

완료코드, 테스트, smoke 또는 정적 gate가 현재 저장소 기준으로 통과했습니다.
부분 구현동작 가능한 골격은 있으나 운영 전체 시나리오 또는 runtime 검증이 남아 있습니다.
미구현요구 기능을 수행하는 코드 경로가 아직 없습니다.
미검증코드나 스크립트는 있으나 이번 작업에서 실행 검증하지 않았습니다.
실패이번 작업에서 실행했고 실패했습니다.
재확인 필요환경 차이나 외부 조건 때문에 별도 판단이 필요합니다.

기능 상태

영역기능상태증거남은 확인
PFW복합 거래 segment 표준완료TransactionSegmentService, TransactionSegmentContext, TransactionSegmentScope, TransactionSegmentMapper.xml, CpfHeaderPropagator운영 부하와 장기 보존 정책 검증은 다음 단계입니다.
PFW SQL복합 거래 구간 로그 테이블완료pfw_transaction_segment, V19__transaction_segment_trace.sql, 99_smoke_check.sql실제 MariaDB 전체 재설치 검증은 별도 smoke로 확인합니다.
ACC/MBR/EXS복합 거래 교육 API부분 구현AccCompositeEducationController, MbrCompositeEducationController, ExsCompositeEducationController, XyzCompositeTransactionEducationControllerruntime smoke는 로컬 서비스 기동 후 실행해야 합니다.
ADM거래 그룹 목록/상세 API부분 구현AdmTransactionGroupController, AdmTransactionGroupService, /adm/api/transaction-groups브라우저 timeline UX와 runtime DB 데이터 조회 검증은 다음 단계입니다.
ADM거래 timeline/header/external logs API부분 구현/segments, /timeline, /headers, /external-logs실거래 데이터 기반 민감정보 마스킹 확인은 runtime smoke에서 확인합니다.
CMN고정길이 전문 registry, group, error, type converter완료FixedLengthLayoutRegistry, FixedLengthGroupSpec, FixedLengthMessageError, FixedLengthTypeConverter, FixedLengthMessageParserFormatterTest반복부 고급 포맷과 전문 사전 DB화는 다음 단계입니다.
문서README와 가이드 현행화완료README.md, 트랜잭션_가이드.html, 표준_헤더_가이드.html, 관리자_가이드.html, SQL_가이드.html최종 정본화 단계에서 HTML 배포본 디자인을 별도 정리할 수 있습니다.
