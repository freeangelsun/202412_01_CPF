CPF 기능 구현 매트릭스

이 문서는 현재 저장소 기준 기능 구현 상태와 검증 상태를 분리해 기록합니다. 실행하지 않은 검증은 완료로 표시하지 않으며, 기존 증적이 있어도 이번 작업에서 재실행하지 않은 항목은 비고에 그 사실을 남깁니다.

상태값은 완료, 부분 구현, 미구현, 미검증, 실패, 재확인 필요만 사용합니다. 최종 판단은 CPF_STABILIZATION_REPORT.md의 검증표와 반드시 일치해야 합니다.

상태 기준

완료코드, 테스트, smoke 또는 evidence로 현재 작업 범위에서 확인한 상태입니다.
부분 구현핵심 구조는 있으나 runtime, UI, 권한, 감사, 외부환경 검증 중 일부가 남은 상태입니다.
미구현요구 기능을 수행하는 코드 경로가 아직 없는 상태입니다.
미검증구현 또는 기존 증적은 있으나 이번 작업에서 실행 검증하지 않았거나 필요한 runtime 환경이 닫혀 있는 상태입니다.
실패검증 명령을 실행했고 실패 결과가 evidence로 남은 상태입니다.
재확인 필요결과 해석 또는 범위 판단을 다시 확인해야 하는 상태입니다.
