CPF 기능 구현 매트릭스

이 문서는 기능별 구현 상태를 실제 파일, API, DB, Swagger, EDU, 테스트 증거와 함께 대조합니다. 부분 구현과 미검증 항목은 완료로 표시하지 않습니다.

기능별 상태

영역기능상태주요 증거검증남은 리스크
PFW거래 ID와 표준 헤더구현TransactionIdGenerator, TransactionContextFilter, LoggingAspect컴파일, transaction ID 표준 검사실제 운영 로그의 로그일자와 ID 일자 불일치 여부는 DB 데이터로 계속 확인해야 합니다.
PFW표준 응답과 예외부분 구현CpfGlobalExceptionHandler, pfw_response_code, pfw_message단위 테스트와 OpenAPI smoke 필요OpenAPI 공통 오류 schema 품질 검증이 추가 필요합니다.
PFWBatch 공통 API부분 구현CpfBatchLauncher, CpfBatchOperationRepository, pfw_batch_*, BATCH_*ADM 배치 서비스 테스트, runtime smoke 대상장시간 실행 heartbeat listener, ghost 자동 감지, 실 JobRepository 부하 시나리오가 남아 있습니다.
CMN프로젝트 공통 채번/알림/로그부분 구현CmnSequenceService, CmnNotificationLogService, CmnBusinessLogServiceCMN 테스트, 동시성 테스트프로젝트별 정책과 운영 보존기간은 적용 프로젝트에서 확장해야 합니다.
ADM운영자 권한부분 구현AdmApiAuthFilter, AdmPermissionController, adm_role, adm_button컴파일, 정적 UI smoke서버 권한 차단 통합 테스트와 버튼 비활성 사유 UX 검증이 필요합니다.
ADM회원 관리부분 구현AdmMemberController, AdmMemberOperationService, mbr_member, mbr_member_role정적 UI smoke회원 등록/수정/상태 변경 전체 E2E 검증이 필요합니다.
ADM로그 관제부분 구현AdmLogController, AdmLogQueryService, pfw_transaction_log정적 UI smokeJSON pretty print, 고정길이 전문 포맷팅, 원문 보기 권한 클릭 검증이 필요합니다.
ADM배치 관제부분 구현AdmBatchController, AdmBatchOperationService, PfwBatchScheduler, pfw_batch_*단위 테스트, ADM runtime smoke 대상관계도 그래프, 실제 브라우저 클릭, 장시간 listener 갱신은 추가 고도화 대상입니다.
ADM기동 health와 runtime smoke보강 중AdmHealthController, scripts/smoke-adm-runtime.ps1health, OpenAPI, 배치 API, cleanup 검증브라우저 클릭 런타임이 없으면 클릭 검증은 미검증으로 남깁니다.
BIZADM업무 관리자 기본 구현부분 구현BizAdmAuthController, BizAdmOperationController서비스 테스트Controller부터 DB까지 이어지는 통합 테스트가 추가 필요합니다.
MBR회원 업무 구현부분 구현MbrController, MbrAuthController, MemberMapper서비스/Controller 테스트회원 상태 전환, 탈퇴/복구, 권한 만료 통합 테스트가 필요합니다.
EXS대외연계 구현부분 구현ExsOperationController, ExsFlowController, exs_*서비스 테스트실제 기관 adapter, token refresh, retry worker 검증이 필요합니다.
XYZ EDU교육 샘플부분 구현xyz/src/main/java/cpf/xyz/edu컴파일, OpenAPI smoke배치 유형별 샘플과 라인 단위 교육 주석은 계속 보강해야 합니다.
SQL/Flyway설치와 migration부분 구현specs/sql, Flyway V1 이후 migrationSQL 표준 검사, MariaDB smoke실제 DB 재실행 결과가 최신 리포트에 남아야 합니다.
Swagger/OpenAPIAPI 문서기동 검증 필요Controller annotation, PfwOpenApiAutoConfigurationOpenAPI smoke앱 미기동 상태의 정적 marker만으로 성공 처리하지 않습니다.
