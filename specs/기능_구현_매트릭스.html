CPF 기능 구현 매트릭스

이 문서는 현재 저장소 기준 구현 상태와 검증 상태를 분리해 기록합니다. 실행하지 않은 검증은 완료로 표시하지 않습니다.

상태값은 완료, 부분 구현, 미구현, 미검증, 실패, 재확인 필요만 사용합니다.

상태 기준

완료코드, SQL, 테스트, smoke, evidence가 현재 저장소 기준으로 확인된 상태입니다.
부분 구현동작 경로는 있으나 운영 UI, 실환경, broker, 권한 감사 등 일부가 남은 상태입니다.
미구현요구 기능을 수행하는 코드 경로가 아직 없는 상태입니다.
미검증코드나 스크립트는 있으나 이번 작업에서 실제 검증하지 않았거나 환경이 없어 실행하지 못한 상태입니다.
실패검증을 실행했고 실패한 상태입니다.
재확인 필요요구 범위나 외부 조건이 애매해 별도 판단이 필요한 상태입니다.

기능 상태

영역기능상태증거남은 확인
PFW표준 헤더와 확장 헤더 전파완료scripts/smoke-standard-header-e2e.ps1, standard-header-e2e-result.sanitized.json브라우저 화면에서 헤더 탭 클릭 검증은 별도 도구가 필요합니다.
PFW복합 거래 segment trace완료pfw_transaction_segment, TransactionSegmentService, TransactionSegmentMapper.xml운영자 UX의 tree 시각화는 계속 고도화 대상입니다.
ACC/MBR/EXS복합 거래 성공/실패 교육 API완료scripts/smoke-composite-transaction-runtime.ps1, scripts/smoke-composite-transaction-failure-runtime.ps1실제 외부기관 adapter는 다음 단계입니다.
ADM거래 그룹 목록, 상세, timeline, headers, external logs완료/adm/api/transaction-groups, scripts/smoke-adm-transaction-group-runtime.ps1브라우저 클릭 자동화는 미검증입니다.
EXS송수신 원장 저장과 ADM 원장 우선 조회완료exs_transaction_log, exs_message_log, EXS_TRANSACTION_LOGtimeout, retry, circuit breaker 실운영 시나리오는 다음 단계입니다.
CMNfixed-length parser, formatter, DB 사전완료FixedLengthLayoutRegistry, cmn_fixed_length_layout, cmn_fixed_length_fieldADM 사전 관리 화면은 다음 단계입니다.
EDU/XYZMapper DB slice 교육 fixture완료xyz_edu_query_fixture.sql, CPF_XYZ_EDU_MAPPER_DB_USERNAME, CPF_XYZ_EDU_MAPPER_DB_USER신규 교육 샘플 추가 시 fixture 이름을 표준으로 유지해야 합니다.
문서README, 가이드, 기능 매트릭스, 안정화 리포트 정합성완료scripts/check-html-docs.ps1, scripts/check-feature-evidence.ps1최종 정본 단계에서는 Markdown 원본 기반 문서 체계를 별도 정리합니다.
