CPF 기능 구현 매트릭스

현재 저장소 기준 구현 상태와 검증 상태를 분리해서 기록합니다. 실행하지 않은 검증은 완료로 표시하지 않습니다.

상태값은 완료, 부분 구현, 미구현, 미검증, 실패, 재확인 필요만 사용합니다.

상태 기준

완료코드, SQL, 테스트, smoke 또는 정적 gate가 현재 저장소 기준으로 통과했습니다.
부분 구현동작 가능한 골격은 있으나 운영 전체 시나리오, 화면 클릭 검증, 실 broker 검증 등 일부가 남아 있습니다.
미구현요구 기능을 수행하는 코드 경로가 아직 없습니다.
미검증코드나 스크립트는 있으나 이번 작업에서 실행 검증하지 않았습니다.
실패이번 작업에서 실행했고 실패했습니다.
재확인 필요환경 차이, 외부 조건, 검증 기준 변경 때문에 별도 판단이 필요합니다.

CMN고정길이 전문 registry, group, error, type converter완료FixedLengthLayoutRegistry, FixedLengthGroupSpec, FixedLengthMessageError, FixedLengthTypeConverter, FixedLengthMessageParserFormatterTest반복부 DB 사전과 전문 layout 관리 UI는 다음 단계입니다.
