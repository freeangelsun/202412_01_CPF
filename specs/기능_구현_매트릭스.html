기능 구현 매트릭스

CPF 기능을 소스, API, 화면, DB, Swagger, EDU 샘플, 검증 결과 기준으로 대조하기 위한 HTML 검수표입니다.

상태 기준

  • 완료: 소스와 문서가 있고 로컬 검증까지 성공한 항목입니다.
  • 일부 구현: 코드 또는 SQL은 있으나 앱 기동, 브라우저 클릭, 실 DB, 외부 broker 검증이 남은 항목입니다.
  • 미구현: 설계 또는 후보만 있고 실제 운영 구현이 없는 항목입니다.
  • 미검증: 환경 부재 또는 미실행으로 결과를 확인하지 못한 항목입니다.
