CPF 기능 구현 매트릭스

이 문서는 기능별 구현 상태, 증거 파일, 검증 기준, 남은 리스크를 한눈에 보기 위한 검수용 문서입니다. 완료는 실제 파일과 테스트 증거가 있을 때만 사용합니다.

상태 값 기준

상태의미
완료소스, SQL, 문서, 검증 증거가 현재 저장소 기준으로 존재합니다.
부분 구현핵심 구조는 있으나 운영 시나리오, UI, 자동화, 외부 연동 일부가 남았습니다.
미구현요구 기능의 실체가 아직 없습니다.
미검증구현은 있으나 이번 작업에서 실제 검증을 실행하지 않았습니다.
실패검증을 실행했고 실패했습니다.
재확인 필요환경 의존 또는 요구 해석이 남아 추가 확인이 필요합니다.

핵심 기능 상태

영역기능상태증거남은 리스크
PFW거래 ID, 거래 헤더, 표준 응답, 예외, 로그완료TransactionIdGenerator, TransactionContextFilter, LoggingAspect운영 부하 기준 성능 테스트는 별도입니다.
PFWBatch 공통 API와 CPF Batch 메타부분 구현CpfBatchLauncher, CpfBatchOperationRepository, pfw_batch_*listener heartbeat와 ghost 자동 감지는 다음 보강 후보입니다.
BATstandalone 배치 worker완료BatApplication, BatBatchRepositoryConfig, BatSmokeJobConfig실제 운영 배치 Job 확장은 업무별로 필요합니다.
BATruntime smoke미검증scripts/smoke-bat-runtime.ps1이번 작업에서 실행한 결과를 안정화 리포트에 따로 기록합니다.
ADM운영자/권한/회원/로그/배치/캐시/메시지/코드/설정부분 구현adm/src/main/java/cpf/adm/opr/controller, adm/src/main/resources/static/adm/adm.js브라우저 실제 클릭 자동화와 세밀한 UX 검증이 남았습니다.
CMN프로젝트 공통 채번, 알림, 업무 로그완료CmnSequenceService, CmnNotificationLogService, CmnBusinessLogService프로젝트별 확장 정책은 도입 프로젝트에서 확정합니다.
MBR회원 CRUD, 인증, token hash, 이력완료MbrController, MbrAuthService, MemberMapper실제 SSO/MFA 연동은 별도입니다.
EXS외부 연계 운영 API와 로그완료ExsOperationController, ExsFlowService실제 외부기관 연동 테스트는 별도 환경이 필요합니다.
XYZ EDU교육 샘플부분 구현xyz/src/main/java/cpf/xyz/eduBAT 유형별 샘플과 center-cut 샘플 보강이 필요합니다.

문서 상태

문서상태역할주의사항
README완료짧은 진입점과 문서 링크상세 절차를 과도하게 넣지 않습니다.
specs/index.html완료문서 읽는 순서와 모듈 책임신규 문서가 생기면 링크를 갱신합니다.
specs/배치_개발_가이드.html완료BAT 개발 기준Job/Step 패턴이 늘면 EDU 샘플과 함께 보강합니다.
specs/운영_매뉴얼.html완료구동, 중지, 장애 조치, smoke 절차운영 자동화가 늘면 runbook을 함께 갱신합니다.
specs/관리자_가이드.html완료ADM 운영 기능 기준브라우저 UX 검증 결과는 리포트로 분리합니다.
specs/SQL_가이드.html완료DB 소유권, 설치 SQL, batch metaSQL 변경 시 설치본과 migration을 함께 갱신합니다.

검증 기준

검증명령완료 판단
HTML 문서scripts/check-html-docs.ps1specs 문서와 안정화 리포트가 HTML 구조를 만족합니다.
UTF-8/mojibakescripts/check-utf8.ps1 -CheckMojibake대상 텍스트 파일이 UTF-8 no BOM이고 주요 mojibake marker가 없습니다.
기능 증거scripts/check-feature-evidence.ps1핵심 파일과 문서 marker가 존재합니다.
Gradle test.\gradlew.bat test --offline모듈 테스트가 성공합니다.
qualityGate.\gradlew.bat qualityGate --offline문서, SQL, UTF-8, 기능 증거 검증이 묶여 통과합니다.
