# CPF 기능 구현 매트릭스

## 검수 메모

- 이 매트릭스는 현재 소스 기준의 보수적 판정입니다.
- `일부 구현`은 소스 또는 SQL은 있으나 앱 기동, 브라우저 클릭, 실제 DB 적재, OpenAPI JSON 검증 중 하나 이상이 부족한 상태입니다.
- 기능을 보강할 때는 이 문서의 `주요 소스`, `API`, `ADM 화면`, `DB 테이블`, `EDU 샘플`, `테스트`를 함께 갱신해야 합니다.
