CPF 로컬 개발환경 구성 가이드

이 문서는 신규 개발자가 로컬 PC에서 CPF를 빌드하고, MariaDB 설치 SQL을 적용하고, ADM/BAT/XYZ를 기동해 smoke 검증까지 수행하는 순서를 정리합니다.

실행하지 않은 항목은 성공으로 기록하지 않습니다. DB 접속, 앱 기동, 브라우저 클릭 검증은 실제 실행 결과가 있을 때만 완료로 표시합니다.

빌드와 정적 검증

컴파일은 JDK 선택과 관계없이 --release 21로 고정합니다. Java 21은 배포 호환 기준이고 Java 25는 개발·테스트 호환 기준이므로, 도구 체인을 변경할 때 두 JDK의 cleanTest qualityGate를 모두 확인합니다. Java 25 호환 기준은 Lombok 1.18.46과 Byte Buddy/agent 1.18.7입니다.

의존성이 이미 내려받아진 환경에서는 offline 검증을 먼저 실행합니다. 네트워크가 필요한 신규 의존성은 별도 승인 후 online 빌드로 확인합니다.

.\gradlew.bat :pfw:test --offline
.\gradlew.bat :adm:test --offline
.\gradlew.bat :bat:test --offline
.\gradlew.bat test --offline
.\gradlew.bat qualityGate --offline
검증명령완료 기준
UTF-8와 글자 깨짐scripts/check-utf8.ps1 -CheckMojibake검사 스크립트가 성공 종료합니다.
HTML 문서 구조scripts/check-html-docs.ps1specs 하위 문서가 HTML 구조이고 불필요한 Markdown 조각이 없습니다.
기능 증거scripts/check-feature-evidence.ps1핵심 Controller, Service, SQL, EDU, 문서 증거가 존재합니다.
SQL 표준scripts/check-sql-standard.ps1테이블 prefix, 공통 감사 컬럼, COMMENT, index 이름 기준을 통과합니다.

MariaDB 설치 SQL

로컬 설치는 분리 SQL을 직접 순서대로 실행하거나, 합본 SQL을 실행합니다. 합본은 SOURCE 명령이 아니라 실제 SQL 본문을 포함합니다.

mysql --ssl=0 --default-character-set=utf8mb4 -u root -p < specs/sql/00_all_install.sql
mysql --ssl=0 --default-character-set=utf8mb4 -u root -p < specs/sql/00_all_install_and_smoke.sql
파일용도주의
specs/sql/00_all_install.sqlDB 생성, 권한, schema, seed, test data 합본 설치로컬 초기화 성격이 있으므로 기존 데이터 보존이 필요한 DB에 실행하지 않습니다.
specs/sql/00_all_install_and_smoke.sql설치 후 smoke query까지 포함FK, index, seed idempotent, 샘플 데이터 조회 결과를 확인합니다.
specs/sql/migration/flyway기존 DB 증분 반영용 migration신규 baseline과 기존 DB 업그레이드 경로를 분리합니다.

앱 기동

PFW와 CMN은 라이브러리 모듈입니다. 실제 기동 대상은 ADM, BAT, XYZ, 업무 서비스 모듈입니다.

.\gradlew.bat :adm:bootJar --offline
.\gradlew.bat :bat:bootJar --offline
.\gradlew.bat :xyz:bootJar --offline

java -jar adm/build/libs/adm-0.0.1-cpf.adm-SNAPSHOT.jar
java -jar bat/build/libs/bat-0.0.1-cpf.bat-SNAPSHOT.jar
java -jar xyz/build/libs/xyz-0.0.1-cpf.xyz-SNAPSHOT.jar
모듈기본 포트확인
ADM8090운영 콘솔, Swagger, 로그/배치/캐시 관제
BAT설정 기준Spring Batch worker와 health API
XYZ8099EDU 샘플 API와 조회 MyBatis 샘플
